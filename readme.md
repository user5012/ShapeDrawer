## Getting Started

### Prerequisites

- Java 17+ installed
- Gradle installed (or use the Gradle wrapper)

### Clone the Repository

```bash
git clone https://github.com/user5012/ShapeDrawer.git
cd shape-drawer
```

### Build the Project

```bash
gradle build
```

### Run the Project

```bash
gradle run
```
